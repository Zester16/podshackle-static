# PodShackle

Podshackle is a podcast app designed to play podcast from various sources


## Tech Stack
Vanila JS
HTML
CSS

## How to create branch
- if its a feat, create feat/V_0.0.version-what-you-are-doing
- example feat/V_0.0.10-add-play-list: this means its feature, with version we are working on is 10 and we are trying to add a play list functionality

## How to add comments
- to add comments it should be v_0_0_yourVersion: what you will be doing
- for example: v_0_0_11A: add reuters morning bid
